Bienvenido a la aplicacion

¿Qué hace nuestra aplicación?

Nuestra aplicación se encargará de cargar cualquier tipo de url o realizar cualquier búsqueda

¿Cómo funciona?

En nuestra aplicación se pueden realizar 2 tipos de búsqueda, búsqueda mediante url o búsqueda especifica.
Puedes ver un videotutorial con el funcionamiento aquí

Buscar mediante url

Para buscar mediante url tienes que introducir una url funcional en el input de página, cuando hagas esto al pulsar en visitar cargará la página web. Para buscar no hace falta que introduzcas el https:\\, aunque igualmente es recomendable

Buscar especifica

Debes seleccionar el tipo de búsqueda que quieras hacer en el desplegable, una vez hecho esto introduciras lo que quieras buscar en el campo de debajo.
